# Project Pumpkin

This project connects hotels around calicut and displays the details of the hotels to the user. It shows hotels 
around the user's location based on google location api. uses nosql firebase databse for storing data. Include search option.

## Getting Started

The main entry point file is app.js.
run it on node and listen on localhost:3000 or the port set.

### Prerequisites

your pc must have node js installed.
dependencies include express.js..
